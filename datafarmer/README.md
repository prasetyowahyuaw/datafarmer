# Datafarmer Package
this is the python package for several data related job

## How to install
run this command in your terminal
```
pip install git+https://github.com/prasetyowahyuaw/datafarmer.git#subdirectory=datafarmer
```
## How to contribute
1. this package utilizing poetry as package manager. you need to install poetry first within your local environment. you can use this command
    ```
    pip install poetry
    ```

## How to test
1. create a test function within the `tests` directory.
2. to run all the test function, make sure you already entry this directory in the terminal. then please run this command
    ```
    pytest
    ```
